tmux, Vim and BASH configs.

My BASH/Vim/tmux configs are organized into modularized directories that make it a lot easier to find what I'm looking for.

Installing

  1. Clone the repo into your home directory
  2. git submodule init && git submodule update
  3. Run ./install (This will move any previous .dotfiles to .file.old (.bashrc -> .bashrc.old for example)
  4. Important: Update your ~/.gitconfig otherwise you'll be committing as me. You don't want that.
  5. Install the powerline font https://powerline.readthedocs.org/en/latest/installation/linux.html#font-installation

Using

  • For tmux it's setup to have GNU screen keybindings, that is, ctrl-a is the leader key, not ctrl-b.
  • The BASH setup is as follows: aliases and functions are in ~/.bash/ and are separated by purpose into different files in ~/.bash/alias_lib/ and ~/.bash/func_lib/ respectively.
  • Input mode in BASH is set to vi
  • See ~/.vim/bundle/ for the Vim plugins I have installed
